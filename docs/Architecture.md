# Creator OS Architecture

Creator OS is an internal web application for managing content production workflows.

## Tech Stack

- Backend: FastAPI
- Frontend: Jinja2 + Bootstrap 5
- Database: SQLite
- ORM: SQLAlchemy
- Deployment: Windows local first, Ubuntu VPS later

## Core Modules

- Dashboard
- Video Projects
- Keywords
- SEO
- Analytics
- Automation

## Architecture Principle

The project is built as a modular system so it can support YouTube first and other platforms later.
